Agent Skill 项目

暗黑职场

卡牌驱动的叙事生存职场模拟器

Python · SQLite · LLM

01 为什么做这个？

职场是什么？赚钱的地方？实现自我价值的地方？都对，但不全对。

真正的职场是——一个没人给你说明书的游戏。

  • 你以为在打工，其实在打没有存档点的通关游戏
  • 老板说"再扛一下"、HR 找你"聊聊"、同事说"顺手帮一下"——懂的都懂
  • 怎么懂的？被坑多了就懂了

暗黑职场的目标：在被坑之前，先在游戏里体验一遍被坑的感觉。
这叫——先崩溃，才能真正躺平。

02 方案：聊天即玩的生存模拟器

  • 玩家扮演职场新人，面对甩锅、压榨、站队、绩效规训
  • 通过事件卡触发抉择，每步都有代价
  • 目标不是"赢"，是别死得太快
  • Agent 形态：飞书/微信/钉钉 → 打开就能玩

"周五上线的两周需求" — 硬扛？甩锅？留痕？摸鱼？每个选择当下结算，有些代价像信用卡账单，下个月才知道有多痛。

03 技术实现：已落地 8 个模块

Python 3.12 SQLite TypeScript Agent Skill

runtime/db.py

SQLite 持久化：5 张表 + 迁移，回合结算自动落盘

runtime/engine.py

回合结算引擎：状态机、d20 判定、隐患、项目、时间段

runtime/materials.py

素材库 + 自定义卡牌：支持 md/txt/pdf 导入

scripts/distill_template.py

卡牌蒸馏器：AI 从素材自动生成角色卡/事件卡/隐患卡

04 设计亮点：机制即表达

d20

随机判定系统

还原职场的不可控性，运气也是实力的一部分

COR

污染值 (Corruption)

甩锅自保赢了当下，输了灵魂——黑化有代价

T+1

延迟爆炸机制

今天的捷径是明天的坑，隐患比即时伤害更可怕

边界

策略来自边界管理

留痕、缩小范围、锁定版本——不是打赢，是别输
