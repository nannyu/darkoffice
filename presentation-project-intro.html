01 问题：职场游戏都在"骗"你

市面上的职场游戏

努力 → 变强 → 升职 → 巅峰。把职场包装成打怪升级，暗示"付出就有回报"。

真实的职场

努力 → 背锅 → 加班 → 体检异常 → 升职的是会汇报的那个。

我们需要一个"诚实"的职场模拟器：不教你成功，教你在注定有点糟的系统里，怎么活得不那么糟。

02 方案：聊天即玩的生存模拟器

  • 玩家扮演职场新人，面对甩锅、压榨、站队、绩效规训
  • 通过事件卡触发抉择，每步都有代价
  • 目标不是"赢"，是别死得太快
  • Agent 形态：飞书/微信/钉钉 → 打开就能玩

"周五上线的两周需求" — 硬扛？甩锅？留痕？摸鱼？每个选择当下结算，有些代价像信用卡账单，下个月才知道有多痛。
