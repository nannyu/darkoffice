### 工作内容
- 爬取 https://www.ccdi.gov.cn/jzn/index.html 全部9页共120条案例
- 其中20条为图文案例（ccdi.gov.cn域名），100条为视频案例（v.ccdi.gov.cn域名，无法提取正文）
- 20条图文案例中成功抓取19条正文（1条"搭圈子勾兑利益"被验证码拦截）
- 所有19条已导入素材库，分类为"反腐案例"，标签含"警钟""中央纪委""违纪违法"

### 技术要点
- 直接 requests 高频请求会触发验证码拦截，需8秒间隔才能绕过
- web_fetch 工具可稳定绕过验证码抓取正文
- 列表页用 requests 即可抓取（无验证码），详情页需慢速或 web_fetch
- ccdi.gov.cn 有两种内容路径：/jzn/（警钟频道）和 /yaowenn/（要闻频道）

### 相关文件
- `scripts/crawl_jingzhong.py`: 列表页爬取（requests + BeautifulSoup）
- `scripts/fetch_and_import_jingzhong.py`: 详情页抓取+素材库导入（含延时策略）
- `data/crawl_results.json`: 原始爬取结果（120条）
- `data/jingzhong_cases.json`: 图文案例元数据（20条）

### 导入结果
- 素材ID 1-19，全部为"中央纪委国家监委网站-警钟"来源
- 平均内容长度约4000字符，含完整案例正文

## 女娲式蒸馏器升级 (2026-04-21)

### 升级内容
- **distill_template.py 全面重写**：融入女娲 Skill 的深度分析框架
  - 新增六维素材分析模板（权力结构/人物画像/事件链/制度漏洞/心理博弈/可游戏化）
  - 角色卡模板新增：mind_model（心智模型）、decision_heuristics（决策启发式）、inner_tension（内在张力）、anti_pattern（反模式）、vulnerability（脆弱点）
  - 事件卡模板新增：structural_trap（结构性陷阱）、hidden_cost（隐藏代价）、hazard_hook（隐患钩子）
  - 隐患卡模板新增：origin_story（起源故事）、explosion_scene（爆炸场景）、chain_reaction（连锁反应）、intervention_chance（干预机会）
  - 剧情线模板新增：theme（核心主题）、power_arc（权力弧光）、每幕 power_shift（权力变化）
  - Schema 校验同步扩展支持所有新字段
- **新增 distill_from_material.py**：智能蒸馏工作流脚本
  - `analyze`: 输出六维分析提示词
  - `character/event/hazard/storyline`: 基于分析结果生成各类卡牌提示词
  - `full`: 一键输出完整工作流
  - `write`: 将 JSON 卡牌数据写入数据库并可选激活
  - 自动截断过长素材（8000/6000字符阈值）
  - 自动生成递增的自定义卡牌 ID

### 设计原则（女娲式）
- 角色不是标签堆砌，是"可运行的认知操作系统"（心智模型+决策启发式+表达DNA+反模式+内在张力）
- 事件不是"发生了什么事"，是"结构性困境中被迫做选择"
- 隐患是"当时觉得没事后来爆炸"的定时炸弹，有起源/爆炸/连锁/干预四要素
- 剧情线是"权力关系的演变史"，有起承转合+角色弧光
