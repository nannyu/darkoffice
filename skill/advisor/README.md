# 职场参谋模块

> 暗黑职场的职场参谋（工具模式）核心文件索引

## 文件结构

```
skill/advisor/
├── darkoffice-advisor-skill.md   # 技能入口（主文件）
├── fengvideo_core.md             # 峰video思维框架（核心）
└── response_templates.md         # 对话模板库
```

## 角色定位

**职场老炮顾问**：整合峰video的126万粉职场内容+302个视频经验，为用户提供实战型职场建议。

## 核心能力

1. **情境定位**：快速判断问题本质（权力博弈/责任边界/晋升发展/人际冲突）
2. **选项生成**：3-4个选项，每个选项带短期/中期/风险/峰video点评
3. **后果预测**：最好/最坏情况 + 概率分布 + 半年后可能
4. **个人建议**：综合分析给出参考建议

## 思维框架

- 6大心智模型（信号理论、契约分工、过客主义、信息不对称防御、工具化思维、反常识判断）
- 8条决策启发式（重复武器论、防御=犹豫=自残、音量降70%等）
- 草台班子分类学（局部bug型/中层瘫痪型/上层失能型/全面崩塌型）

## 触发机制

用户描述职场困境时激活，触发词：
- 领导/同事/晋升/站队/背锅/甩锅/加班/绩效
- "帮我分析下""有什么建议""该怎么办"

不触发：
- "开始游戏""新游戏" → 娱乐模式

## 使用方式

当用户进入工具模式时：
1. 先读取 `darkoffice-advisor-skill.md` 了解完整规则
2. 参考 `fengvideo_core.md` 调用思维框架
3. 使用 `response_templates.md` 生成标准话术
