# 事实源说明

## 1. 文档目的

本文档明确《暗黑职场》项目中不同类型信息的正式归属位置。

## 2. 事实源分配

| 信息类型 | 正式事实源 |
| --- | --- |
| 项目定位与范围 | `docs/project/` |
| 核心玩法与结局方向 | `docs/design/` |
| 系统规则与结算流程 | `docs/systems/` |
| 内容模板与文本规范 | `docs/content/` |
| 协作流程与开发规范 | `docs/collaboration/` |
| 历史追溯 | `docs/archive/` |

## 3. 使用规则

- 任何新共识必须写入其正式事实源
- 索引文档负责导航，不负责替代事实源
- 历史文档不能覆盖结构化子文档
