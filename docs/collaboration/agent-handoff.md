# AI/Agent 交接规范

## 1. 文档目的

本文档规定 AI/Agent 在接手或交接《暗黑职场》项目任务时应遵循的最小规则。

## 2. 接手任务前最小读取集

1. `README.md`
2. `docs/INDEX.md`
3. `docs/collaboration/development-guidelines.md`
4. 对应领域的事实源文档

## 3. 交接摘要格式

```markdown
## 交接摘要

- 已完成：
- 已更新文档：
- 关键决定：
- 未解决问题：
- 下一步建议：
```

## 4. 交接原则

- 明确说清修改了什么
- 明确引用修改依据
- 明确哪些内容仍是待定
- 若发现冲突，优先提醒，不继续叠加修改
