# 结局方向

## 1. 文档目的

本文档定义《暗黑职场》第一版的主要结局方向与结局设计原则。

## 2. 第一版主要结局方向

当前建议保留 4 个大方向：

1. 猝死 / 崩溃
2. 被开除
3. 苟活离职
4. 黑化上位

## 3. 结局触发逻辑

### 3.1 崩溃结局

典型触发：

- `HP <= 0`
- 长期低 EN 与高压状态叠加
- 重大暴雷与透支共同引爆

### 3.2 被开除结局

典型触发：

- `KPI <= 0`
- 问责链与组织态度恶化叠加
- HR、上司与机制卡形成联合压力

### 3.3 苟活离职结局

典型触发：

- 玩家维持基本生存线
- 没有彻底黑化
- 完成阶段性求生目标后离场

### 3.4 黑化上位结局

典型触发：

- `COR` 长期偏高
- 通过权谋、甩锅、灰色合作获得组织收益
- 以牺牲底线为代价获得稳定位置

## 4. 后续扩展方向

后续可扩展：

- 证据反杀
- 精神独立
- 同流合污
- 派系牺牲品

## 5. 设计原则

- 结局不只是数值终点，应体现价值代价
- 好结局不应完全无伤
- 坏结局不应只有惩罚，也要能回看是如何一步步形成
- 结局必须和玩家长期选择模式相关，而不只是最后一次判定
