# 角色名录

## 1. 文档目的

本文档记录《暗黑职场》第一版推荐角色池及其系统职责，作为内容生产入口。

## 2. 第一版角色池

| 角色 | 系统定位 | 典型压力 |
| --- | --- | --- |
| 黑心老板 | 最高层价值绑架者 | 大局、成长、牺牲 |
| PUA 上司 | 高频施压核心 | 加班、问责、模糊边界 |
| 笑里藏刀老油条 | 日常隐患制造者 | 口头承诺、经验压制 |
| 推活同事 | 横向边界侵蚀者 | 顺手帮忙、任务转移 |
| 派系总监 | 政治压力放大器 | 站队、表态、汇报 |
| 关系户 | 规则例外制造者 | 沟通失效、项目失误 |
| 财务 / 采购关键人 | 合规与灰区入口 | 流程、报销、倒签 |
| 乙方供应商 | 外部协同失真源 | 口头答应、推诿、压线 |
| 甲方金主 | 时间与交付压迫源 | 加急、变更、临时需求 |
| HR 笑面虎 | 制度性追责者 | 绩效、态度、制度沟通 |

## 3. 内容使用原则

- 每个角色至少要有一个稳定被动
- 每个角色至少要有一类高频偏好事件
- 每个角色的说话风格应稳定
- 同一角色可以复杂，但不能失去可识别性

## 4. 后续扩写要求

- 扩写角色前，先补对应人物卡模板
- 新角色若与现有角色职责重叠，必须说明差异
- 派系角色必须说明其利益结构与站队价值
