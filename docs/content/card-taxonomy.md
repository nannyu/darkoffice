# 卡牌分类法

## 1. 文档目的

本文档用于统一《暗黑职场》的卡牌分类视角，帮助内容生产时避免混淆系统职责。

## 2. 主分类

- 人物卡
- 事件卡
- 应对卡
- 状态卡
- 隐患卡
- 机制卡
- 项目卡

## 3. 事件卡子分类

- 工作任务类
- 甩锅类
- 情绪压榨类
- 政治站队类
- 灰色合规类
- 甲方需求类
- 绩效制度类
- 恢复机会类

## 4. 应对卡子分类

- 直接执行
- 留痕防御
- 拖延规避
- 讨价还价
- 转移责任
- 明确拒绝
- 恢复自保
- 组织政治动作

## 5. 使用规则

- 一张卡必须先确定主分类，再考虑标签
- 子分类用于内容组织，不替代正式结算规则
- 若新增子分类，必须说明它与现有分类的区别
