# 设计支柱

## 1. 文档目的

本文档定义《暗黑职场》的长期设计支柱，用于约束后续系统扩写、内容生产与原型取舍。

## 2. 设计支柱

### 支柱 1：压力优先，爽感其次

《暗黑职场》不是英雄幻想游戏。玩家的快感来自险中求活、勉强自保和看清结构，而不是一路碾压反派。

### 支柱 2：短期收益必须伴随长期代价

几乎所有“当回合很好用”的选择，都应该在后续回合留下代价：

- 熬夜硬扛 -> 透支
- 甩锅自保 -> 污染上升
- 公开站队 -> 长期埋雷
- 灰色合作 -> 风险累积

### 支柱 3：隐患比即时伤害更可怕

真正致命的不是当回合被压一次，而是那些被暂时压下去、之后翻面引爆的问题。

### 支柱 4：角色坏，但要坏得合理

每个角色都应有利益动机、惯用手法和稳定口吻，而不是脸谱化地吼叫。

### 支柱 5：制度比个人更坏

个人是具体施压者，制度、流程、评价体系和权力结构才是压迫感的底层来源。

### 支柱 6：策略来自边界管理，而不是暴力对抗

玩家的高质量应对，通常不是“打赢对面”，而是：

- 留痕
- 缩小范围
- 锁定版本
- 转化责任边界
- 延迟冲突
- 找到更可承受的代价

## 3. 使用规则

- 新机制若削弱以上支柱，应重新评估
- 新内容若只增加戏剧性、不增加结构压力，不应纳入核心体系
- 若未来技术实现与设计支柱冲突，以设计支柱为准
