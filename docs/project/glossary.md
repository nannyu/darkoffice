# 术语表

## 1. 文档目的

本文档统一《暗黑职场》项目中的核心术语，避免在不同文档中使用不同说法指向同一概念。

## 2. 项目层术语

| 术语 | 定义 |
| --- | --- |
| 暗黑职场 | 项目中文名 |
| Dark Office | 项目英文名 |
| 卡牌驱动叙事生存 | 项目的核心类型组合，强调叙事推进、系统压迫与资源求生 |
| 文档先行 | 当前阶段原则，指在正式开发前先沉淀规则、结构与协作共识 |
| 事实源 | 某类信息的唯一正式归属文档 |

## 3. 玩法层术语

| 术语 | 定义 |
| --- | --- |
| 回合 | 系统的基础推进单位，每回合代表 20 分钟职场时间 |
| 核心循环 | 时间推进、施压、应对、结算、后果累积的基础玩法闭环 |
| 长期压力 | 不在当回合完全结清，而会持续累积或延迟爆发的压力 |
| 链式事件 | 由一次处理结果引发的后续状态、隐患、项目或新事件链 |
| 现实映射 | 将游戏内结算与现实职场逻辑关联起来的提示文本 |

## 4. 数值层术语

| 术语 | 定义 |
| --- | --- |
| `HP` | 生命值，总体承受极限 |
| `EN` | 精力值，最核心的即时消耗资源 |
| `ST` | 体力值，生理能量与恢复基础 |
| `KPI` | 绩效值，组织对玩家的容忍度 |
| `RISK` | 风险值，潜在问题与暴雷概率的累积指标 |
| `COR` | 污染值，玩家对恶性职场规则的内化程度 |
| 阈值状态 | 当某项数值进入特定区间时触发的系统性修正 |

## 5. 卡牌层术语

| 术语 | 定义 |
| --- | --- |
| 人物卡 | 本回合施压来源 |
| 事件卡 | 本回合核心冲突 |
| 应对卡 | 玩家处理冲突的行动方式 |
| 状态卡 | 持续效果载体 |
| 隐患卡 | 延迟爆炸问题载体 |
| 机制卡 | 阶段性全局环境修正 |
| 项目卡 | 持续任务压力载体 |
| 牌区 | 卡牌在界面或系统中的功能区域 |

## 6. 角色与关系术语

| 术语 | 定义 |
| --- | --- |
| 来访角色 | 本回合触发事件的施压角色 |
| 角色态度 | 某角色对玩家的敌意、满意度或好感变化 |
| 派系 | 角色所属的利益阵营或组织立场 |
| 被盯上 | 一类状态，表示玩家更容易成为压迫目标 |

## 7. 协作层术语

| 术语 | 定义 |
| --- | --- |
| 历史文档 | 旧版总案与系统文档，仅用于追溯与迁移 |
| 结构化子文档 | 按主题域拆分后的正式事实源文档 |
| 文档迁移 | 将历史文档中的稳定内容拆入新文档体系 |

## 8. 术语使用规则

- 正式文档优先使用本表术语
- 不为同一概念频繁更换说法
- 若新增核心术语，必须先更新本表再进入其他文档
